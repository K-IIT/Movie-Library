# Movie Library (Личная кинотека)
**Автор:** [Бирюкова Надежда Сергеевна]
**Язык:** Python 3.8+
**Лицензия:** MIT
## Описание программы
Movie Library — это простое приложение с графическим интерфейсом для хранения информации о фильмах.
Позволяет добавлять фильмы с указанием названия, жанра, года выпуска и рейтинга.
Реализована фильтрация по жанру и году выпуска. Все данные сохраняются в файл `movies.json`.
Для интерфейса используется стандартная библиотека `tkinter`.

## Как запустить программу

### 1. Установка Python и зависимостей
*   **Python:** Убедитесь, что у вас установлен Python версии 3.8 или новее. Скачать можно с официального сайта `python.org`.
*   **Зависимости:** Проект использует только стандартную библиотеку Python (`tkinter` входит в её состав), поэтому устанавливать дополнительные пакеты не требуется.
*   *Проверка установки:* Откройте терминал и введите `python --version` или `python3 --version`.

### 2. Запуск приложения

**Вариант А: Из исходного кода**
1.  Клонируйте репозиторий на свой компьютер:
    ```bash
    git clone <ссылка_на_репозиторий>
    cd movie_library
    ```
2.  Запустите скрипт из папки проекта:
    ```bash
    python movie_library.py
    # или для некоторых систем:
    python3 movie_library.py
    ```
3.  Окно приложения должно появиться на экране.
