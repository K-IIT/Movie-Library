# Movie Library (Личная кинотека)
**Автор:** [Бирюкова Надежда Сергеевна]
**Язык:** Python 3.8+
**Лицензия:** MIT
## Описание программы
Movie Library — это простое приложение с графическим интерфейсом для хранения информации о фильмах.
Позволяет добавлять фильмы с указанием названия, жанра, года выпуска и рейтинга.
Реализована фильтрация по жанру и году выпуска. Все данные сохраняются в файл `movies.json`.
Для интерфейса используется стандартная библиотека `tkinter`.
